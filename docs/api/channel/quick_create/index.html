Definition

POST /wp/v2/YOUR_APP_PN/quick-channel

Parameters

参数 说明
domains 域名列表
backends 原站列表
mc_origin_id MC源ID

入参样例

{
  "domains": [                                  // Required
    {
      "domain": "xxx.xxx.xxx",                  // Required 域名
      "cache_rules": [                          // Optional 缓存规则
        {
           "orig_path": "/*",                   // Required 具体的缓存规则
           "max_age"  : 600 or "no-cache",      // Required 缓存时间或不缓存
           "order_num": 1                       // Required 缓存规则优先级，数字小优先级高
        },
        ...
      ]
    }
  ]
  "backends": [                                // Optional 不能和mc_origin_id同时为空
    {
      "backend": "domain or ip",               // Required 源站 域名格式或ip格式 域名有且仅有一个 ip格式可以多个
      "port"   : 80,                           // Optional 默认80
    },
    ...
  ],
  "mc_origin_id": 1                            // Optional MC源ID 不能和backends同时为空   
}

Example

请求方法

curl -X POST \
  http://dev.api.tan14.cn/wp/v2/<YOUR_APP_PN>/quick-channel \
  -H "Content-Type: application/json" \
  -H "Tan14-Key: <YOUR_TAN14-KEY>" \
  -d "{\"domains\":[],\"backends\":[]}"
