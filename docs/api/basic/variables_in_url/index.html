  • URL中的变量

在 TAN14 CDN 的 API 定义中， 部分接口的URL地址中是包含资源占位符的。 资源占位符在URL中以 大写的斜体 字符出现 现将常见的几个资源占位符做个简要介绍。

占位符名称 含义 常见获取位置
YOUR_APP_PN 您在TAN14 CDN的产品ID 在TAN14 App Console 后台账户 页面获取
YOUR_APP_CHANNEL_ID 您在TAN14 CDN建立的某个频道的ID 在TAN14 App Console 后台设置 页面获取
YOUR_APP_DOMAIN_ID 您在TAN14 CDN的某个频道中添加域名所对应的ID 从 查询单个频道信息 和 添加域名接口均可以获取
YOUR_APP_BACKEND_ID 您在TAN14 CDN的某个频道中添加源站所对应的ID 从 查询单个频道信息 和 添加源站 接口均可以获取
YOUR_APP_MCBACKEND_ID 您将某一TAN14 CDN网络存储实例绑定到频道后产生的ID 从 查询单个频道信息 接口返回的 mcbackends.id 即为该值
YOUR_APP_MCBACKEND_INS_ID 您在TAN14 CDN创建的网络存储实例的ID 从 查询名下已开通网络存储列表 可以获取
YOUR_APP_MD5_TASK_ID 您在使用缓存校验功能时候会获取到的ID 提交缓存校验任务 会获得该ID
YOUR_APP_PRELOAD_ID 您在添加一条预加载任务时会获取到的ID 加载任务) 会获得该ID
YOUR_APP_REFRESH_ID 您在添加一条刷新任务时会获取到的ID 添加刷新任务 会获得该ID
