  • 返回状态码
状态码 说明
200 OK 成功返回，无错误
201 资源创建成功
400 请求参数错误，详见返回数据中的 detail 字段
401 需要提供Tan14-Key做认证，或者提供的值已过期
403 当前Tan14-Key对应的用户没有调用该接口的权限
404 资源没有找到
405 方法不允许
500 服务器错误， 遇见该错误请联系我们的管理员
503 服务器临时不可用
