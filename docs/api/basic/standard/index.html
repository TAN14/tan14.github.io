测试与生产地址

为了您能够在浏览该文档的同时， 可以方便的使用页面上提供的 API 测试工具 来测试某个接口， 文档中默认的API地址是由我们提供 的模拟测试环境， 用来协助您快速的理解和掌握我们的API， 同时我们配备了一个测试的前端环境, 测试地址：

  • API http://dev.api.tan14.cn
  • 后台 http://dev.app.tan14.cn

待您熟悉了我们的API之后， 您就可以使用我们的生产地址来进行您的业务开发了, 生产地址：

  • API https://api.tan14.cn
  • 后台 https://app.tan14.cn

请求

所有TAN14-API 仅接受 Content-Type 为 application/json 的请求 且请求体格式需为JSON类型 所有请求需要携带 Tan14-Key 头进行身份验证

参数

TAN14 CDN 的API 包含两种参数，一种是 URL中的占位符 ， 另一种是请求携带的参数，定义在 Parameters 区域， 其发送方式遵循标准的 RESTful 标准： 即 GET请求通过 QueryString 发送， 其他方式通过 http body 的方式来发送。

响应

所有TAN14 API接口返回数据类型均为 JSON 类型
