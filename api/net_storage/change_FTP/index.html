修改网络存储的FTP信息

请求地址

PUT 

http://dev.api.tan14.cn/wp/v2/YOUR_APP_PN/mc/backend/instance/YOUR_APP_MCBACKEND_INS_ID/ftp
携带参数
ftp_user:string     修改后的ftp帐号

ftp_passwd:string   修改后的ftp密码
响应
返回字段说明
字段 说明
id TAN14 CDN网络存储实例ID, URL占位符中的 MCBACKEND_INS_ID 的真实值
mc_origin_id 创建时指定的TAN14网络存储
ftp_user ftp帐号
ftp_passwd ftp密码
cname 该存储实例的地址
pn 所属产品(即您自己的PN)
rsync rsync同步方式信息, 具体参考 为TAN14网络存储开通rsync同步
created 创建时间
