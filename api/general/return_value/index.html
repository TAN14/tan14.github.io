  • 常见返回
状态码 意义
200 接口成功返回
201 资源创建成功
400 请求参数错误，详见返回数据中的 detail 字段
401 需要提供Tan14-Key做认证，或者提供的值已过期
403 当前Tan14-Key对应的用户没有调用该接口的权限
404 资源没有找到
405 方法不允许
500 服务器错误， 遇见该错误请联系我们的管理员
503 服务器临时不可用

提示

关于接口返回400错误， 多数是由于用户提供的参数不合法导致， 具体原因参见接口返回JSON数据中的detail键所描述的原因
